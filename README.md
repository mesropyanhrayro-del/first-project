# Python story generator(Mad Libs)
A python program that generates fun and silly stories based on user input.
The user can choose a story template or let the program choose randomly,
if the user input does not meet the project conditions, fill in words and then receive a generated story.

## Features
Choose from 3 story templates or selected one randomly.

## Templates
1. **Hospital Story**
2. **Camping Story**
3. **Enchanted Castle Story**

Each template asks for different types of words (nouns, verbs, adjectives etc.) and inserts them into a story.
